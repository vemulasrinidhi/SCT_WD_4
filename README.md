# 📝 To-Do Web Application  

An interactive To-Do List web application built using HTML, CSS, and JavaScript. This project allows users to manage daily tasks efficiently with a modern UI and real-time updates.  

---

## 🚀 Features  
- ✅ Add new tasks  
- ✏️ Edit tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as completed  
- 📊 Task statistics (Total, Completed, Pending)  
- 🎨 Dark theme with gradient UI  
- ⚡ Real-time updates  

---

## 🛠️ Technologies Used  
- HTML – Structure  
- CSS – Styling and design  
- JavaScript – Functionality  

---

## 🎯 How to Use  
1. Enter a task in the input field  
2. Click **Add** button  
3. Manage tasks using:  
   - ✔️ Complete  
   - ✏️ Edit  
   - ❌ Delete  
4. View task stats
